Installation
============

``pip install awpy``

To install awpy, you must have Go (the programming language) installed. You need a version greater than 1.17, and you can visit this link to install the latest `Golang version <https://go.dev/dl/>`_.

Once you have Golang installed, you can check your version in command line by using the command ``go version``. After this, simply run ``pip install awpy``.
